# 个人简历网页

这是一个简洁大气的个人简历网页，采用现代化的设计风格，展示个人经历和联系方式。

## 技术栈

- HTML5
- CSS3 (使用 Flexbox 和 Grid 布局)
- JavaScript (原生)
- 响应式设计

## 特点

- 简洁大气的设计风格
- 流畅的动画效果
- 响应式布局，适配各种设备
- 优化的性能表现

## 项目结构

```
personalpage/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript 交互文件
└── README.md       # 项目说明文件
```

## 使用说明

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 文件即可查看效果
3. 根据个人需求修改 `index.html` 中的内容
4. 可以通过修改 `styles.css` 来自定义样式
5. 在 `script.js` 中可以添加或修改交互效果

## 浏览器支持

- Chrome (推荐)
- Firefox
- Safari
- Edge
